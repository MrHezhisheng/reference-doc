mbot的python接口
=====================

本章介绍了程小奔的mbot接口，一共包含以下几类模块:

* mbot的python接口：主要是指mbot板载驱动的一些API接口。
* 第三方类库的python接口：mbot内置的一些第三方类库的接口类，例如mqtt、urequest的类。
* 神经元扩展模块的python接口：mbot扩展神经元模块时的一些API接口。
